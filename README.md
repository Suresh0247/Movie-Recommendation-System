# Movie Recommendation System

This project recommends movies based on similar genres using Machine Learning.





## Features

- Movie recommendation system
- Genre similarity matching
- Interactive Streamlit app
- Beginner-friendly ML project



## Technologies Used

- Python
- Pandas
- Scikit-learn
- Streamlit



## Machine Learning Concept

This project uses:
- CountVectorizer
- Cosine Similarity

to find similar movies.
